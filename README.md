## Prérequis - Structure :

1. Installer chaque dépendance : `npm install`
2. Base de donnée de démo : `bdd_demo/react_todo_task`
3. Server :
  - Démarrage : `npm start`
  - Adresse du client : `http://localhost:4000/`
  - pages :
    - tasklist : `/tasks`
    - tasklist : `/addtasks`
    - tasklist : `/deletetasks`
4. Client :
  - Démarrage : `npm start`
  - Adresse du client : `http://localhost:3000/`
  - pages :
      - tasklist : `/`
